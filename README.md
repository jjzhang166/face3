人脸识别项目
============

###项目依赖项：

	`OpenCV`，版本不限
	`cxxlib`，地址[http://github.com/tkorays/cxxlib](http://github.com/tkorays/cxxlib)，也可以使用bitbucket的secret版本
	`TinyXML`，ext目录里已有，无需下载，每次编译需要加入其cpp文件。
	

###说明

* 主要类放在face目录，可以点击进入，查看。（内附文档）
* 其他vs项目是一些demo，包括face文件xml管理，识别的例子等。
